### 安装

***1.ubuntu14.04安装jekyll***

sudo apt-get install ruby

sudo apt-get install nodejs

sudo apt-get install jekyll

sudo apt-get install ruby-redcarpet

sudo apt-get install ruby-kramdown

***2.mac**

brew install jekyll

### 使用

jekyll server -I
